# 😴 Sleep Quality Predictor

A Machine Learning web application that predicts sleep quality based on user lifestyle inputs like sleep duration and stress level.

## 🚀 Features
- Predicts sleep quality (Good / Average / Poor)
- Simple user interface using Flask
- Machine Learning model (Decision Tree)

## 🛠 Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas

## ▶️ How to Run
1. Install dependencies:
pip install pandas scikit-learn flask

2. Train model:
python train.py

3. Run app:
python app.py

## 📊 Output
Users input sleep duration and stress level → model predicts sleep quality.
